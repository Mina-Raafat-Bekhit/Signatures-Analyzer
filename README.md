# Signatures-Analyzer
Developed a Java-based file analyzer that identifies the authentic format of audio and image files by parsing their binary data and their File Signatures.
# How it works
Implemented binary stream processing using FileInputStream to extract and convert byte headers into Hexadecimal format for signature verification (e.g., detecting FLAC, WAVE, BMP, and GIF).
Integrated a graphical file explorer interface using JFileChooser to provide a user-friendly way for selecting and analyzing files.
